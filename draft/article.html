Percentage increase formula

Percentage increase measures how much a value has grown compared with its original amount. It is useful whenever you have a clear starting value and a newer, larger value.

The result expresses the increase as a percentage of the original value, which is why the original value is always the denominator.

Formula
((New Value − Original Value) / Original Value) × 100
Original valueThe starting number before the increase.
New valueThe number after the increase.
ResultThe growth rate expressed as a percent.
Useful shortcut

Convert the original and new values to the same unit before calculating. For example, do not compare dollars with cents or kilograms with pounds.

How to calculate percentage increase step by step

You can calculate the change reliably in three simple steps. Keep the original value nearby because you will use it twice.

1

Find the increase

Subtract the original value from the new value to find the amount of growth.

2

Compare it with the original

Divide the increase by the original value, not by the new value.

3

Convert to a percentage

Multiply the decimal result by 100 and add the percent sign.

Example calculation

Suppose a monthly subscription changes from $50 to $65. The price increased by $15, but the percentage increase tells us how large that $15 change is relative to the original price.

From $50 to $65Worked example
Original value$50
New value$65
Difference$65 − $50 = $15
Divide by original$15 / $50 = 0.30
Multiply by 1000.30 × 100 = 30%

The subscription price increased by 30%. You can verify this result or calculate a different change with the percentage calculator.

Common percentage increase mistakes

The formula is short, but choosing the wrong denominator or mixing up related percentage concepts can produce a misleading result.

Do not divide by the new value

Percentage increase describes growth relative to where you started, so divide the difference by the original value.

  • Using the new value as the denominator. This understates the size of the increase.
  • Forgetting to multiply by 100. A decimal of 0.30 is equal to 30%, not 0.30%.
  • Confusing increase with percentage points. A rate moving from 20% to 30% rises by 10 percentage points but increases by 50%.
  • Using zero as the original value. Percentage increase from zero is undefined because division by zero is impossible.

Percentage increase vs. percentage difference

Use percentage increase when one number is clearly the original. Use percentage difference when the two values are peers and neither is a natural starting point.

ConceptFormulaBest used for
Percentage increase(New − Original) / Original × 100Growth from an old value to a new value
Percentage decrease(Original − New) / Original × 100A drop from an old value to a new value
Percentage differenceDifference / Average × 100Comparing two values with no clear original
Percentage pointsNew rate − Original rateDescribing the absolute change between percentages

When should you use percentage increase?

Percentage increase makes changes easier to compare across values of different sizes. A $10 increase can be tiny or enormous depending on the starting amount.

  • Tracking revenue, profit, traffic, or audience growth over time
  • Measuring a price increase or cost change
  • Comparing test scores, productivity, or performance improvements
  • Understanding investment growth before accounting for time or compounding
  • Reporting changes in population, inventory, or demand

Can the increase be more than 100%?

Yes. An increase above 100% means the new value is more than double the original. For example, a value rising from 40 to 100 increases by 150% because the increase of 60 is one and a half times the original 40.

Frequently asked questions

Quick answers to the most common questions about calculating percentage increases.

What is the formula for percentage increase?
Subtract the original value from the new value, divide the result by the original value, and multiply by 100: ((New − Original) / Original) × 100.
Do I divide by the original value or the new value?
Divide by the original value. Percentage increase measures the size of the change relative to the starting amount.
Can percentage increase be more than 100%?
Yes. If the new value is more than twice the original value, the percentage increase will be greater than 100%.
What if the original value is zero?
A percentage increase from zero is mathematically undefined because the formula would require division by zero. Report the absolute change instead.
How do I calculate percentage decrease?
Subtract the new value from the original value, divide by the original value, then multiply by 100.
