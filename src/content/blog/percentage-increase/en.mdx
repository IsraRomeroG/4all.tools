Percentage increase tells you how much a value grew compared with its original value. It is useful for prices, traffic, revenue, grades, measurements, and any situation where the starting value matters.

## Percentage increase formula

Use this formula:

```text
percentage increase = ((new value - original value) / original value) x 100
```

The original value is the baseline. The new value is the number after the change.

## Example

Imagine a value rises from 80 to 100.

```text
((100 - 80) / 80) x 100 = 25%
```

The increase is 25 percent because the value grew by 20, and 20 is one quarter of the original 80.

## Common mistakes

Do not divide by the new value when you are measuring increase from an original value. That answers a different question.

Do not use percentage increase when the original value is zero. There is no valid baseline, so the result is undefined.

Use percentage difference when you want to compare two values without treating either one as the starting point.
